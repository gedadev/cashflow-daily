# Cashflow Daily app

This is a React app to manage your daily cash balance, with a expense tracking feature and pending payments control, to calculate your daily earnings.

## Tech Stack

### Frontend:

- React + Vite: UI Library and build tool.
- Tailwind CSS: For styling.
- Lucide React: Icon library.

### Backend & Infrastructure:

- Hono: The web framework used to build the API.
- Cloudflare Workers: The serverless platform where the backend runs.
- Cloudflare D1: The SQL database used for storing data.
- TypeScript: The language used across the entire stack for type safety.

### Environment:

- Node.js: The environment used for local development and tooling.

## Features

- Daily balance tracking
- Expense tracking
- Pending payments control
- Daily earnings calculation
